# Prismaticallization DC日版汉化工具

## 概述

本脚本工具用于将DC日版游戏Prismaticallization修改翻译为简体中文版。翻译内容使用AI生成。

## 使用方法

### 准备工作

- 安装Python 3。
- 使用pip安装Pillow。
- 找到Prismaticallization的DC日版镜像，必须是GDI格式。如果格式不同请自行转换。

### 使用本工具

- 克隆本项目。
- 修改项目根目录中main.py的 ROM_SRC / ROM_OUT / PATH_SRC / PATH_OUT ，以分别指向你的源镜像和输出镜像地址。
- 运行main.py。
- 如果运行完成没有错误，汉化完成的文件会在你设置的PATH_OUT中，格式也为GDI。
